# domain/ — Design

Pure domain types with no external dependencies beyond the standard library.

## Principles

- Types use `db` struct tags for sqlx scanning and `json` tags for API serialization.
- UUIDs are strings in Go — generated by PostgreSQL, parsed/validated at boundaries.
- Status fields use typed string constants for compile-time safety.
- Filter types encapsulate query parameters for list operations.
- `json.RawMessage` for the `tags` field to allow flexible JSON arrays without a concrete type.
